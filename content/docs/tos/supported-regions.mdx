## 1. 平台部署位置

<Callout type="error" title="服务器位于海外，不提供中国大陆直连节点">
  **ProAPI 所有的边缘网关、控制台、API 入口与日志存储均部署在海外服务器**。我们 **没有** 在中国大陆境内部署任何本地化服务节点，**也不计划** 在可预见的未来部署。

  我们不是中国境内法律意义上的「生成式人工智能服务提供者」，也不向中国境内公众提供本地化的 AI 服务。
</Callout>

## 2. 上游厂商支持地区限制

ProAPI 作为聚合层，调用的所有模型能力来自第三方上游厂商。这些厂商对其服务的可用地区有 **各自独立** 的限制：

| 上游厂商 | 中国大陆支持情况 | 来源 |
| --- | --- | --- |
| **OpenAI** | ❌ 不支持，IP 限制 | [openai.com/supported-countries](https://help.openai.com/en/articles/5347006-openai-api-supported-countries-and-territories) |
| **Anthropic** | ❌ 不支持；2025-09 起进一步禁止中国控制的公司 | [anthropic.com/supported-countries](https://www.anthropic.com/supported-countries) |
| **Google (Gemini)** | ❌ 中国大陆不在 Gemini API 可用地区 | Google Cloud 区域文档 |
| **xAI (Grok)** | ⚠️ 视具体接口而定 | xAI 官网 |

<Callout type="warn" title="跨境访问的本质">
  当您从一个上游厂商不支持的地区（如中国大陆）使用 ProAPI 时，请求会先到达 **ProAPI 海外服务器**，再由我们转发至上游厂商。**上游厂商看到的是 ProAPI 服务器的 IP，不是您的本地 IP**。

  这一架构使技术调用得以完成，但 **不能消除您在本地司法管辖区下的访问合规义务**。
</Callout>

## 3. 中国大陆用户必读

如果您从中国大陆访问本服务，**请在使用前完整阅读本节并自行评估法律风险**。

### 3.1 相关法规

<Accordions>
<Accordion title="《生成式人工智能服务管理暂行办法》（2023-08-15 施行）">

由国家互联网信息办公室等七部门联合发布，核心条款：

- **第二条** — 未向境内公众提供生成式人工智能服务的，不适用本办法。
- **第十七条 / 备案** — 提供具有舆论属性或社会动员能力的生成式 AI 服务，应当依法履行 **大模型备案** 或 **算法备案**。
- **第二十条** — 对来源于境外向境内提供生成式人工智能服务不符合规定的，国家网信部门可通知有关机构采取技术措施和其他必要措施予以处置。
- **第二十一条** — 违反本办法的，按《网络安全法》《数据安全法》《个人信息保护法》《科学技术进步法》等处罚；情节严重的，可责令暂停服务、追究刑事责任。

</Accordion>
<Accordion title="《数据安全法》《个人信息保护法》（PIPL）">

- 重要数据 / 个人信息出境，需通过 **数据出境安全评估** 或签订 **个人信息出境标准合同**。
- 通过本平台向境外模型发送含个人信息的请求，构成 **数据出境** 行为。
- 数据出境合规义务由 **数据处理者（即您）** 承担。

</Accordion>
<Accordion title="《互联网信息服务深度合成管理规定》">

对生成的图像、视频应当依规作 **AI 生成内容标识**（显式标识与隐式水印）。

</Accordion>
</Accordions>

### 3.2 用户自负条款

<Callout type="error" title="使用本服务即视为已知悉以下事项">
  1. **本服务不针对中国大陆境内公众提供** — 您从中国大陆访问本服务的行为，不构成本平台向境内公众提供服务。
  2. **访问工具合规自负** — 您使用何种网络访问工具（包括但不限于 VPN、代理）访问本平台，由您自行评估并承担全部法律责任。
  3. **数据出境合规自负** — 您将含个人信息或重要数据的内容通过本服务发送至境外模型，由您自行履行数据出境合规义务。
  4. **生成内容备案自负** — 若您将通过本服务获得的输出向中国境内公众提供，您应自行依据相关法规完成备案 / 安全评估 / 内容标识。
  5. **上游政策风险自负** — 上游厂商可能随时收紧对中国大陆用户的检测与限制（如 Anthropic 2025-09 政策），由此导致的服务可用性变化由用户自行承担。
</Callout>

### 3.3 平台不承担的责任

<Callout type="warn" title="责任划分">
  对于以下因您本地法律法规导致的后果，**ProAPI 不承担任何责任**：

  - 您因使用未经备案的境外 AI 服务而被监管机关采取的措施
  - 您因数据出境未履行合规义务而引发的行政处罚
  - 您本地访问网络工具的合法性争议
  - 您将生成内容用于中国境内公众且未完成备案产生的责任
  - 上游厂商政策变更导致的账号或服务不可用
</Callout>

## 4. 其他受限地区

参照美国与欧盟出口管制制度，本服务 **不向以下地区** 提供：

- 受美国制裁的国家与地区
- 朝鲜
- 俄罗斯（受最新制裁限制范围内）
- 伊朗
- 古巴
- 叙利亚
- 乌克兰受军事控制区域（克里米亚、顿涅茨克、卢甘斯克、扎波罗热、赫尔松）

若用户从上述地区访问本服务并产生违规后果，由用户自行承担。
