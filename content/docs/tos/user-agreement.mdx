## 摘要

本平台为您提供基于各大人工智能模型的 **接口中转聚合服务**。使用本服务即表示您承诺以合法、合规的方式使用模型能力，**严禁** 用于生成和传播任何违法敏感信息（尤其是违规图像与视频）。您需妥善保管接口密钥，并了解 **预付费余额不支持无理由退款** 的规则。

<Callout type="warn" title="平台定位声明">
  ProAPI 是 **技术中转聚合层**，不开发底层模型、不持久化业务内容、不对生成结果做实质性内容审查。所有模型能力均由上游第三方厂商（OpenAI / Anthropic / Google 等）提供。**用户作为内容请求与使用的最终责任方，须独立承担调用模型所产生的全部法律与合规后果**。
</Callout>

</Callout>

### 2.2 图像与视频生成专项限制

针对平台提供的多模态模型（如图像绘制、视频生成模型），您必须遵循以下 **极其严格** 的合规要求：

<Callout type="error" title="零容忍清单">

  - **严禁儿童剥削与低俗色情** — 对儿童性虐待材料（CSAM）及任何形式的色情、淫秽视觉内容采取「零容忍」政策。

### 2.3 违约处理

若我们的自动化风控系统或人工抽查发现您的请求涉及上述违规行为，我们有权 **在不事先通知的情况下**：

1. 立即 **永久封停** 您的账户；
2. **清零所有余额**，不予退还；
3. 将相关日志数据移交国家有关执法机关依法处理；
4. 保留进一步追究您民事 / 刑事责任的权利。

## 3. 账户安全与密钥管理

您需要通过邮箱或受信任的第三方授权完成注册。注册成功后，您将在控制台获取用于调用服务的接口密钥。

### 密钥的唯一性

接口密钥是您调用模型和计费的唯一凭证。您必须将其 **视同银行密码般妥善保管**：

- **严禁** 硬编码在公开的客户端代码（如前端网页、移动端本地应用）中；
- **严禁** 提交到公共代码仓库（GitHub / GitLab / Gitee 等）；
- **严禁** 在群聊、论坛、截图中明文展示。

### 连带责任

<Callout type="warn" title="密钥使用全部由您承担">

  若发现密钥疑似泄露，请立即前往控制台使旧密钥作废并重新生成。**密钥泄露后产生的额度损失，平台不予赔付**。
</Callout>

## 4. 计费规则与退款政策

本平台采用 **预付费结合按量计费** 的方式提供服务。

### 4.1 计费规则

您的账户余额将在每次成功的请求后实时扣减。每次请求的扣费标准基于：

- 不同模型设定的「计算倍率」；
- 实际消耗的输入 / 输出字数额度（Token）；
- 或生成的图像 / 视频数量。

您可以在控制台的「模型广场」中查看实时的计费费率表。

### 4.2 退款说明

<Callout type="warn" title="预付费不支持无理由退款">
  由于 API 接口调用的数字化及即时性特征，**充值后的账户余额原则上不支持提现或退款**。
</Callout>

仅在以下例外情况下，可以联系我们协商退款：

1. 因平台方发生重大系统灾难，**导致服务不可用状态连续超过 72 小时**；
2. 支付网关发生明确的技术错误导致的 **重复扣款**。

除上述两种情形外的所有退款请求（包括但不限于：误充值、模型生成效果不满意、上游厂商接口变更、第三方客户端不兼容、用户操作失误等），均不在退款范围内。

## 5. 数据隐私与日志处理

我们高度重视并保护您的数据隐私。作为一个纯粹的中转加速平台，我们的数据流转机制如下：

### 5.1 无状态转发（不存内容）

您的请求（包含提示词、上下文）以及上游模型的响应（生成的文本、图像）**仅在服务器内存中进行高速转发**。我们 **绝不会** 在平台的数据库中持久化存储您的业务数据。

### 5.2 请求日志保留

为了实现精准计费、高频防刷及排查故障，我们会记录必要的请求元数据：

- 请求时间戳
- IP 地址
- 调用的模型名称
- 消耗的计算额度
- 状态码

这些元数据通常仅 **保留 30 天后自动滚动清除**。

### 5.3 协助执法调查

尽管我们不保留生成内容本身，但若收到具有法律效力的警方调查令或法院传票，我们将依法 **提供涉嫌违规用户的注册信息及调用日志**。

> 完整数据处理规范见 [《隐私政策》](/tos/privacy-policy)。
