## GPT / Codex 系列分组

两个 GPT 分组都面向 Codex CLI，区别在于号池来源与倍率。两者均为 Codex 官方渠道，稳定性都为站内最高档。

<Callout type="info" title="怎么选">
- 追求**纯血 Pro 质量**：选 `Codex-Pro`（仅 Pro 账号池）
- 追求**性价比**（日常编程首选）：选 `Codex-Plus-Pro 混池`（Pro + Plus 混池，价格更低）
- 各分组模型列表为编写时快照，最终以**模型广场**实时列表为准。
</Callout>

<Callout type="warn" title="2026-05-25 分组调整">
原 `Codex-Plus`（×0.08）已下线，新分组 `Codex-Plus-Pro 混池`（×0.12）替代——号池由 Codex 官方 Pro + Plus 混合组成，稳定性更高。如果你的令牌仍指向 `Codex-Plus`，请前往控制台编辑令牌切换分组。
</Callout>
