## 什么时候选这个分组

只要你要做下面这些事，就选 `音视频模型`：

- 文生图、图生图、图片编辑。
- 文生视频、图生视频。
- 使用 `nano-banana`、`gpt-image-2`、`seedance`、`kling`、`sora`、`veo` 等模型。

当前默认倍率：`x1`。音视频模型多数是按次计费，VIP / Premium 是否另有折扣请以模型广场为准。

<Callout type="warn" title="不要用 Codex 或 Claude 令牌调画图视频">
图片和视频模型需要 `音视频模型` 分组令牌。用 Codex、Claude、Gemini 分组令牌调用，常见结果是“模型不存在”。
</Callout>

## 调用入口

- 图片生成：`POST /v1/images/generations`
- 图片编辑：`POST /v1/images/edits`
- 视频生成：推荐看 [音视频生成 API](/api/media)

<Callout type="warn" title="视频模型要耐心等">
视频生成通常比聊天慢很多。客户端超时时间建议设置到 5 分钟以上；如果工具支持异步任务，优先使用异步视频接口。
</Callout>
