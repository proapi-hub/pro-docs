## 音视频模型分组

`音视频模型` 用于画图 / 视频 / 音频生成，是站内画图与视频模型的聚合分组。倍率为 `x1`。

<Callout type="info">
本分组模型列表为编写时快照，最终以**模型广场**实时列表为准。
</Callout>

- **分组介绍：**

  - 音视频模型渠道聚合分组，聚合了多个画图 / 视频 / 音频生成渠道。需要生成图片或视频时选择此分组。
  - `grok-imagine-*` 系列已从 `Grok` 分组迁入本分组，按 `x1` 全额计费。
  - 2026-05-23 接入 **OpenArt 聚合渠道**，新增 12 个图像生成模型（Flux 2 / Imagen 4 / Seedream / Qwen Image / Wan 2.7 等），同时 `nano-banana` / `nano-banana-pro` / `gpt-image-2` 三个已有模型增加 OpenArt 作为备用渠道，混池路由。

- **计费倍率：** `x1`（VIP `x1` · Premium `x1`，**全等级不打折**，以保障音视频上游成本）

- **稳定性：** ⭐⭐⭐

- **适用 CLI / 工具：**

  - API 调用（画图 / 视频生成接口）

- **是否支持接入第三方：**

  - ✅ 支持

- **包含模型（编写时快照）：**

  - **画图（原有）**：`nano-banana`、`nano-banana-2`、`nano-banana-pro`、`gpt-image-2`、`imagen-4`、`grok-imagine-image`、`grok-imagine-image-lite`、`grok-imagine-image-pro`、`grok-imagine-image-edit`
  - **画图（2026-05-23 OpenArt 新增）**：
    - Flux 系列：`flux-2-pro`、`flux-2-max`
    - Google：`google-imagen-4`
    - 字节跳动 Seedream：`byte-plus-seedream-4`、`byte-plus-seedream-4-5`、`byte-plus-seedream-5-lite`
    - 阿里通义 Qwen Image：`qwen-image-2`、`qwen-image-2-pro`、`qwen-image-plus`
    - 阿里通义万相：`wan2-7-image`、`wan2-7-image-pro`
    - xAI Grok：`grok-imagine`
  - **视频**：`sora-2`、`sora-2-pro`、`veo-3.1`、`veo-3.1-fast`、`veo-3.1-ref`、`kling-3.0`、`kling-o3`、`grok-imagine-video`

<Callout type="info">
不同渠道支持的模型略有差异，且画图 / 视频模型按次计费，调用前请到模型广场确认价格与可用性。
</Callout>

<Callout type="info" title="2026-05-24 单价全面调整">
本分组所有图片 / 视频模型单价已于 2026-05-24 重新校准（国产图像普遍下调 60-90%，部分海外视频按真实成本上调）。视频模型已统一为 **一口价、不限时长**。完整价格快照见 [音视频 API · 参考单价](/api/media#参考单价)。
</Callout>

<Callout type="info" title="原“统一分组”已下线">
原 `统一分组`（站内全模型聚合）已于 2026-05-15 下线，所有用户与令牌已自动并入 `default`。如需使用特定模型家族，请在控制台创建令牌时**选择对应专用分组**（Codex / Claude / Gemini / 音视频）以享受折扣倍率。
</Callout>
