## 音视频模型分组

`音视频模型` 用于画图 / 视频 / 音频生成，是站内画图与视频模型的聚合分组。倍率为 `x1`。

<Callout type="info">
本分组模型列表为编写时快照，最终以**模型广场**实时列表为准。
</Callout>

- **分组介绍：**

  - 音视频模型渠道聚合分组，聚合了多个画图 / 视频 / 音频生成渠道。需要生成图片或视频时选择此分组。
  - `grok-imagine-*` 系列已从 `Grok` 分组迁入本分组，按 `x1` 全额计费。

- **计费倍率：** `x1`（VIP `x0.7` · Premium `x0.5`）

- **适用 CLI / 工具：**

  - API 调用（画图 / 视频生成接口）

- **是否支持接入第三方：**

  - ✅ 支持

- **包含模型（编写时快照）：**

  - 画图：`nano-banana`、`nano-banana-2`、`nano-banana-pro`、`gpt-image-2`、`imagen-4`、`grok-imagine-image`、`grok-imagine-image-lite`、`grok-imagine-image-pro`、`grok-imagine-image-edit`
  - 视频：`sora-2`、`sora-2-pro`、`veo-3.1`、`veo-3.1-fast`、`veo-3.1-lite`、`veo-3.1-ref`、`veo-3.1-i2v`、`veo-3.1-i2v-fast`、`veo-3.1-i2v-lite`、`veo-3.1-interpolation`、`veo-3.1-extend`、`kling-3.0`、`kling-o3`、`grok-imagine-video`

<Callout type="info">
不同渠道支持的模型略有差异，且画图 / 视频模型按次计费，调用前请到模型广场确认价格与可用性。
</Callout>

<Callout type="info" title="原“统一分组”已下线">
原 `统一分组`（站内全模型聚合）已于 2026-05-15 下线，所有用户与令牌已自动并入 `default`。如需使用特定模型家族，请在控制台创建令牌时**选择对应专用分组**（Codex / Claude / Gemini / 音视频）以享受折扣倍率。
</Callout>
