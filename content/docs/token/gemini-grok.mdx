## Gemini 与 Grok 分组

`Ant-Gemini` 面向 Gemini CLI，`Grok` 提供 Grok 全系列模型（含画图 / 视频）。

<Callout type="info">
各分组模型列表为编写时快照，最终以**模型广场**实时列表为准。
</Callout>

<Tabs items={['Ant-Gemini', 'Grok']}>
<Tab value="Ant-Gemini">

- **分组介绍：**

  - 反重力官方 Gemini 中转，逆向 Google Antigravity 得到的 Gemini 模型渠道。提供 Gemini 3 / 2.5 全系列，并包含画图模型。

- **计费倍率：** `x0.5`（VIP `x0.35` · Premium `x0.25`）

- **稳定性：** ⭐⭐⭐

- **适用 CLI / 工具：**

  - Gemini CLI（推荐）
  - Cline 等第三方工具

- **是否支持接入第三方：**

  - ✅ 支持

- **包含模型（编写时快照）：**

  - `gemini-3.1-pro-preview`
  - `gemini-3-pro-preview`
  - `gemini-3.1-flash-lite-preview`
  - `gemini-3-flash-preview`
  - `gemini-3.1-flash-image`
  - `gemini-2.5-pro`
  - `gemini-2.5-flash`
  - `gemini-2.5-flash-lite`

<Callout type="info" title="模型名提示">
逆向渠道的模型名相对官方 API 形式有所变化，配置 Cline 等插件前，请先到**模型广场**确认本分组下的准确模型名，避免出现“模型不存在”问题。
</Callout>

</Tab>
<Tab value="Grok">

- **分组介绍：**

  - Grok 官方渠道中转，倍率全站最低（`x0.005`）。提供 Grok 4.20 全系列（含 reasoning / expert / fast / super 等变体）。`grok-imagine` 画图与视频模型已迁移至 **音视频模型** 分组（按全额 `x1` 计费）。

- **计费倍率：** `x0.005`（VIP `x0.0035` · Premium `x0.0025`）

- **稳定性：** ⭐⭐⭐

- **适用 CLI / 工具：**

  - API 调用
  - 各类第三方客户端

- **是否支持接入第三方：**

  - ✅ 支持

- **包含模型（编写时快照）：**

  - 对话 / 推理：`grok-4.3-beta`、`grok-4.20-auto`、`grok-4.20-fast`、`grok-4.20-expert`、`grok-4.20-0309`、`grok-4.20-0309-reasoning`、`grok-4.20-0309-reasoning-super`、`grok-4.20-0309-non-reasoning`、`grok-4.20-0309-non-reasoning-super`、`grok-4.20-0309-super`

<Callout type="info" title="画图 / 视频请走音视频模型分组">
`grok-imagine-image / image-lite / image-pro / image-edit / video` 系列已统一归到 **音视频模型** 分组，按 `x1` 全额计费，避免 `x0.005` 的 Grok 倍率与按次计费的成本错位。
</Callout>

</Tab>
</Tabs>
