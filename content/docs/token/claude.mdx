## 怎么选 Claude 分组

当前真实可用的 Claude 相关分组有 5 个：

| 分组 | 当前倍率 | 稳定性 | 适合场景 |
| --- | --- | --- | --- |
| `claude-max-满血` | `x1.2` | ⭐⭐⭐⭐⭐ | 重要任务、生产环境、追求最高稳定 |
| `Kiro-Claude-高缓存` | `x0.2` | ⭐⭐⭐⭐ | Claude Code 日常主力，兼顾价格和稳定 |
| `Ant-Claude` | `x0.4` | ⭐⭐⭐ | 普通 Claude 调用、备用线路 |
| `claude-reverse-高缓存` | `x0.1` | ⭐⭐ | 低价备用，不建议单独作为主力 |
| `Windsurf` | `x0.25` | ⭐ | Claude / GPT / Gemini 综合低价线路 |

<Callout type="warn" title="已移除旧分组">
旧文档里的 `claude-max-特价-满血` 当前不在真实分组列表中。请不要再创建或配置这个分组名。
</Callout>

<Tab value="claude-max-满血">

### claude-max-满血

- **倍率：** `x1.2`
- **定位：** Claude 高稳定线路。
- **适合：** 长任务、生产环境、重要代码修改、复杂 Agent 工作流。
- **当前主要模型：**
  - `claude-opus-4-7`
  - `claude-opus-4-6`
  - `claude-opus-4-5-20251101`
  - `claude-sonnet-4-6`
  - `claude-sonnet-4-5-20250929`
  - `claude-haiku-4-5-20251001`

<Callout type="info" title="新手怎么理解">
这是 Claude 里最稳的一档。它不是最便宜，但适合不想频繁换线路的用户。
</Callout>

</Tab>
<Tab value="Kiro-Claude-高缓存">

### Kiro-Claude-高缓存

- **倍率：** `x0.2`
- **定位：** Claude 高缓存线路，当前性价比高。
- **适合：** Claude Code 日常开发、文档理解、翻译、规划任务。
- **当前主要模型：**
  - `claude-opus-4-7`
  - `claude-opus-4-6`
  - `claude-opus-4-5-20251101`
  - `claude-sonnet-4-6`
  - `claude-sonnet-4-5-20250929`
  - `claude-haiku-4-5-20251001`

<Callout type="info" title="推荐顺序">
如果你主要用 Claude Code，又希望价格不要太高，优先试这个分组；遇到稳定性要求很高的任务，再切 `claude-max-满血`。
</Callout>

</Tab>
<Tab value="Ant-Claude">

### Ant-Claude

- **倍率：** `x0.4`
- **定位：** Claude 通用线路。
- **适合：** 普通 Claude 聊天、轻量开发、备用线路。
- **当前主要模型：**
  - `claude-opus-4-7`
  - `claude-opus-4-6`
  - `claude-sonnet-4-6`
  - `claude-haiku-4-5-20251001`

</Tab>
<Tab value="claude-reverse-高缓存">

### claude-reverse-高缓存

- **倍率：** `x0.1`
- **定位：** Claude 低价备用线路。
- **适合：** 价格敏感、可接受偶发失败的任务。
- **当前主要模型：**
  - `claude-opus-4-7`
  - `claude-opus-4-6`
  - `claude-opus-4-5`
  - `claude-sonnet-4-6`
  - `claude-sonnet-4-5`
  - `claude-haiku-4-5`

<Callout type="warn" title="不要只配这一条">
这个分组便宜，但稳定性低于 `Kiro-Claude-高缓存` 和 `claude-max-满血`。建议把它当备用。
</Callout>

</Tab>
<Tab value="Windsurf">

### Windsurf

- **倍率：** `x0.25`
- **定位：** Claude / GPT / Gemini 综合低价线路。
- **适合：** 低价备用、轻量任务、自有服务测试。
- **当前 Claude 相关模型示例：**
  - `claude-sonnet-4.6`
  - `claude-sonnet-4.6-thinking`
  - `claude-sonnet-4.6-1m`
  - `claude-opus-4.6`
  - `claude-opus-4-7-low`
  - `claude-opus-4-7-medium`
  - `claude-opus-4-7-high`
  - `claude-opus-4-7-xhigh`

<Callout type="warn" title="稳定性提示">
Windsurf 模型多、价格低，但不适合作为重要 Agent 工作流的唯一线路。重要任务建议选择 `claude-max-满血` 或 `Kiro-Claude-高缓存`。
</Callout>

</Tab>

## Claude Code 配置提醒

- Base URL 请按对应教程填写，不要把普通 OpenAI `/v1` 地址误填到 Anthropic 原生配置里。
- 如果工具支持 Anthropic 原生协议，优先使用 `/v1/messages`。
- 如果只是 OpenAI 兼容客户端，也可以走 `/v1/chat/completions`，但工具调用和缓存能力可能不如原生协议完整。
