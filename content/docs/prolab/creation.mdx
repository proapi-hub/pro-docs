### 2. 选择模型

下拉选择当前任务使用的模型。模型选项来自你接入的所有上游的合集，**按系列分组**展示。

<Callout type="info" title="价格速查">
鼠标悬停模型可看到单价；完整价格表见 [音视频 API · 参考单价](/api/media#参考单价)。**视频模型为一口价，不分时长**。
</Callout>

### 3. 写提示词

中间大文本框，支持中英文。两个辅助按钮：

| 按钮 | 作用 | 使用场景 |
| --- | --- | --- |
| ✨ **AI 优化** | 将你的简短描述扩写为高质量提示词 | 不知道怎么写 / 想要更专业的描述 |
| 📖 **提示词库** | 从本地 / 同步的提示词库插入 | 复用经典提示词，避免重复造轮子 |

<Callout type="warn" title="提示词最少 3 个字符">
ProAPI 后端要求 prompt 至少 3 个字符，否则会返回 `validation_error`。
</Callout>

### 4. 开始生成

底部「**✨ 开始生成**」按钮点击后：

- 任务立即出现在中部「任务列表」的最前面，状态为「**生成中**」；
- 完成后自动加载图片 / 视频，可点击预览、下载；
- **过程中可继续提交新任务**，多任务并发。

<Callout type="info" title="视频生成耗时">
视频模型实际耗时 150 ~ 400 秒，**请耐心等待**。这是上游模型生成时长，不是 ProLab 的延迟。
</Callout>

## 任务列表（中部）

### 顶部过滤

| 控件 | 功能 |
| --- | --- |
| **「绘图工作台」下拉** | 切换工作空间（如果你创建了多个） |
| **「全部」下拉** | 按类型过滤：全部 / 图片 / 视频 / 失败 |
| **🔍 搜索框** | 按提示词关键字搜索历史任务 |

### 任务卡片

每个任务卡片显示：

- **预览**：图片直接展示，视频显示首帧 + 播放控件；
- **模型徽章**：左下角彩色标签（如 `sora-2-pro`、`gpt-image-2`），点击可复制模型名；
- **类型徽章**：右上角「视频 / 图片」标签；
- **任务元数据**：`ID:51`、`3 分钟前`、`耗时 2 分 43 秒`；
- **提示词预览**：底部显示前两行，鼠标悬停看完整内容。

### 卡片悬停操作

鼠标移到任务卡片上，左上角出现 5 个操作按钮：

| 图标 | 操作 | 说明 |
| --- | --- | --- |
| ⬇ | **下载** | 保存图片 / 视频到本地 |
| ⛶ | **放大预览** | 全屏查看 |
| ⓘ | **信息** | 查看完整提示词、模型、参数、耗时 |
| ⧉ | **复制** | 复制提示词到剪贴板 |
| 🗑 | **删除** | 移至回收站 |

## 右上工具栏

| 按钮 | 功能 |
| --- | --- |
| 👁️‍🗨️ **模糊本页** | 一键模糊所有任务封面（演示 / 录屏时遮挡敏感内容） |
| 👁️ **显示本页** | 取消模糊（回到正常状态） |
| 🗑 **回收站** | 查看 / 恢复 / 永久删除已删任务 |
| 「共 N 个任务」 | 当前工作空间的任务总数 |

<Callout type="info" title="数据存储位置">
所有生成任务（图片缩略图、视频 URL、提示词、参数）保存在浏览器 IndexedDB。**清浏览器缓存会清空历史**，重要作品请及时下载。
</Callout>

## 推荐工作流

### 场景 1：海报 / 营销图

1. 模型选 `nano-banana-pro`（高质量）或 `google-imagen-4`（文字渲染好）；
2. 提示词写明：**主体 + 风格 + 比例 + 色调**；
3. 同时提交 3 ~ 5 个变体（多次点「开始生成」）；
4. 在任务列表中挑最满意的，下载备用。

### 场景 2：短视频素材

1. 模型选 `veo-3.1-fast`（性价比，¥0.60）或 `sora-2`（质量优先，¥1.80）；
2. 提示词限定 4 ~ 8 秒短镜头，描述运镜方式；
3. 提交后**离开等待**，5 分钟左右回来查看；
4. 不满意时改 1 ~ 2 个关键词重新生成（视频成本较高，少试多想）。

### 场景 3：批量风格化处理

1. 准备一批关键词（如 10 个不同主题）；
2. 用 ✨ **AI 优化** 一次性扩写为完整提示词；
3. 模型固定为同一个（如 `byte-plus-seedream-4`，¥0.03）；
4. 依次提交，10 分钟内得到 10 张统一风格的图。

## 常见问题

<Accordions>

<Accordion title="Q1：选择模型下拉是空的？">

说明 ProLab 没有拉取到上游模型列表。请回到 [设置 → 上游配置](/prolab/quick-start)，确认 ProAPI 卡片为绿色激活状态。如果是灰色，重新点「快速接入 ProAPI」导入密钥。

</Accordion>

<Accordion title="Q2：视频生成卡在「生成中」很久？">

视频生成正常需要 150 ~ 400 秒，**这是上游模型耗时**，不是 ProLab 卡顿。如超过 10 分钟仍未完成：

- 检查浏览器开发者工具 → Network 看请求状态；
- 关闭代理 / VPN 排除中断；
- 改用 `veo-3.1-fast`（更快）重试。

</Accordion>

<Accordion title="Q3：能批量提交相同提示词的多个任务吗？">

可以。直接连续点「开始生成」即可，ProLab 不限制并发数。**注意账户余额**，视频模型一次 ¥0.60 ~ ¥5.40。

</Accordion>

<Accordion title="Q4：怎么让生成的图保持同一风格？">

两种方式：
- **固定模型 + 风格关键词**：所有提示词都加 "浮世绘风格" / "电影海报风格" 等限定；
- **图生图**：选支持 Image-to-Image 的模型（如 `qwen-image-2`、`byte-plus-seedream-4-5`），上传参考图。

</Accordion>

<Accordion title="Q5：能改变图片的尺寸 / 比例吗？">

可以。在「选择模型」下方会出现该模型支持的 `aspect_ratio` 选项（如 1:1 / 16:9 / 9:16），点击切换。详细取值矩阵参考 [音视频 API · 取值范围](/api/media#取值范围矩阵)。

</Accordion>

</Accordions>
