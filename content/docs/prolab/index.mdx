## ProLab 是什么

**ProLab 是 ProAPI 官方提供的 AI 模型实验平台。**

它主要用于调试和体验 ProAPI 下的 **生图、视频、对话模型**。  
你可以在浏览器中选择模型、填写提示词、调整参数、发送请求，并直观看到生成结果。

相比直接写 API 请求，ProLab 更适合在正式接入前完成模型验证：

- 测试某个图像 / 视频模型是否可用
- 对比不同模型的生成效果
- 调整提示词、尺寸、比例、生成数量等参数
- 直观看到请求参数与生成结果
- 给团队或客户演示模型能力

## 为什么需要 ProLab

如果你已经确定模型和参数，可以直接使用 OpenAI 兼容 API 接入。  
但在正式接入之前，通常需要先回答这些问题：

- 哪个模型适合当前业务？
- 提示词怎么写效果更稳定？
- 图像比例、尺寸、数量等参数应该怎么填？
- 视频模型的任务状态和结果怎么查看？
- 是否可以先在页面里验证，再把参数迁移到代码中？

ProLab 就是为这些调试场景准备的。

## 相关工具

ProLab 是 ProAPI 官方提供的 **Web 实验平台**，重点是可视化调试模型。  
如果你要在其他场景中使用 ProAPI，可以搭配下面这些工具：

| 工具 | 定位 | 适用场景 |
| --- | --- | --- |
| **ProLab** | **Web 实验平台** | 在浏览器里调试生图、视频、对话模型，直观看到参数与结果 |
| [OpenAI 兼容 API](/api/openai) | API 接入方式 | 在自己的应用、脚本或 SDK 中调用模型 |
| [CC-Switch](/ccswitch) | 桌面配置工具 | 管理 Claude Code / Codex / Gemini CLI 等客户端的 API 配置 |
| [Claude Code / Codex / Gemini CLI](/cli) | 命令行工具 | 在终端中使用模型进行编程、修代码和批处理 |

<Callout type="info" title="令牌说明">
ProLab 推荐使用 **音视频分组令牌**。该分组可调用图像、视频等模型，适合用于 ProLab 的创作工作台。  
更多说明请查看 [令牌分组介绍](/token/intro)。
</Callout>
