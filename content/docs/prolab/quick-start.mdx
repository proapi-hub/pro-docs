## 一图看懂接入流程

```
打开 ProLab  →  顶栏「设置」  →  「快速接入 ProAPI」  →  粘贴密钥  →  选择模型  →  完成
   零安装         默认空配置            一键弹窗         3 字段        多选       开始创作
```

全程不需要写配置文件、不需要选模型 ID，**约 60 秒可完成**。

## Step 1：打开 ProLab 设置页

进入 ProLab 后，点击顶部导航栏的「**设置**」，左侧默认停留在「**上游配置**」标签。

<Callout type="info" title="为什么默认显示「未配置平台」？">
ProLab 默认带了一张 ProAPI 占位卡片（含端点地址 `https://newapi.prorisehub.com`），但**没有预置任何密钥**，所以显示「未配置平台」。下一步导入密钥后即可激活。
</Callout>

### 设置页的 8 个标签速览

| 标签 | 用途 | 客户首次对接是否必看 |
| --- | --- | :---: |
| **上游配置** | 管理 API 提供商（端点 + 密钥） | ✅ 必看 |
| 代理配置 | 走代理 / 自定义网络出口 | ⚪ 国内直连可跳过 |
| 模型测试 | 一键测试某个模型是否可用 | ✅ 推荐 |
| 导入 / 导出 | 备份 / 迁移配置 | ⚪ 多端同步时用 |
| Prompt 设置 | 全局系统提示词 | ⚪ 进阶用户 |
| 提示词同步 | 个人提示词库云同步 | ⚪ 进阶用户 |
| MCP 服务 | 接入 Model Context Protocol | ⚪ 进阶用户 |
| 通用设置 | 界面 / 主题 / 语言 | ⚪ 自定义偏好 |

## Step 3（推荐）：跑一次模型测试

为了确认密钥与模型联通正常，切到左侧「**模型测试**」标签：

{/* 图片占位 · 对应「设置 → 模型测试」标签页面。如有截图请在此插入。建议路径：/assets/image/ProLab/04-model-test.webp */}

![模型测试](/assets/image/ProLab/04-model-test.webp)

1. 在模型下拉选 `nano-banana`（最便宜的图像模型，0.04 元/次）；
2. 点击「测试」，等待 3 ~ 5 秒；
3. 看到绿色「✅ 调用成功 · 耗时 X.X s」即可。

<Callout type="warn" title="测试失败怎么办？">
- **401 / 鉴权失败** → 密钥错误或令牌已过期，回 [ProAPI 控制台](https://newapi.prorisehub.com) 检查；
- **403 / 模型不在分组** → 你的令牌分组不是「音视频模型」，需要新建正确分组的令牌；
- **超时 / 网络错误** → 试试切换 ProLab「代理配置」标签，或检查本地网络。
</Callout>

## 进阶：自定义配置（多上游 / 自建中转）

如果你需要同时接入 **多个上游**（如 ProAPI + 自建 NewAPI + OpenAI 官方），点击右上角「**+ 自定义配置**」：

| 字段 | 说明 |
| --- | --- |
| 配置名称 | 自起一个易记的名字 |
| API 端点 | 完整 URL，如 `https://your-relay.com` |
| API 密钥 | 对应上游的密钥 |
| 协议类型 | OpenAI / Anthropic / Gemini（按上游协议选择） |

<Callout type="info" title="多上游路由">
配置多个上游后，ProLab 创作面板的模型下拉会自动合并展示。同一模型名出现在多个上游时，会按上游的优先级或健康度路由。
</Callout>

## 常见问题

<Accordions>

<Accordion title="Q1：我可以直接编辑配置文件吗？">

可以。ProLab 配置存储在浏览器 `localStorage` 中，但**强烈推荐使用「快速接入」弹窗**，避免手动配置 JSON 时出错。如需迁移，用「设置 → 导入 / 导出」生成 `.json` 文件。

</Accordion>

<Accordion title="Q2：我的密钥安全吗？会上传到服务器吗？">

ProLab 是**纯前端 + 用户自托管**架构，密钥仅保存在你的浏览器 `localStorage`，所有 API 请求**直接从你的浏览器发往 ProAPI 端点**，不经过任何第三方中转。

</Accordion>

<Accordion title="Q3：换电脑后配置怎么迁移？">

两种方式：
- **导入 / 导出**：在「设置 → 导入 / 导出」生成 `.json` 文件，到新设备导入；
- **提示词同步**（如已开启）：登录同一账号自动同步配置 + 提示词库。

</Accordion>

<Accordion title="Q4：「快速接入」弹窗的端点为什么是锁定的？">

为了避免客户误填错误端点，「快速接入」固定使用 `https://newapi.prorisehub.com`。如需自定义端点，点击弹窗右上的「**高级：自定义端点**」链接切换到自由编辑模式，或直接走「+ 自定义配置」入口。

</Accordion>

<Accordion title="Q5：我能同时使用 ProAPI 和 OpenAI 官方密钥吗？">

可以。在「+ 自定义配置」新增 OpenAI 官方配置（端点 `https://api.openai.com`，协议选 OpenAI），ProLab 会在创作面板的模型下拉里同时展示两个上游的模型，按上游优先级路由。

</Accordion>

</Accordions>
