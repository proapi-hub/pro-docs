在配置任意 CLI 客户端之前，请先完成本页的软件下载与环境检查。

## 软件下载

<Tabs items={["Windows","MacOS","Linux"]}>
<Tab value="Windows">

1. 打开 [CC-Switch 的 GitHub Release 页面](https://github.com/farion1231/cc-switch/releases/latest)。
2. 滚动到最下方，选择适合自己系统的安装包。Windows 推荐下载普通 `.msi` 后缀的安装包进行安装。

![](/assets/image/CC-Switch/001.webp)

3. 安装后运行 CC-Switch 主程序，界面如下。

![](/assets/image/CC-Switch/002.webp)

</Tab>
<Tab value="MacOS">

macOS 推荐使用 HomeBrew 安装。打开终端后，依次运行以下命令：

```bash
# 添加 tap 源
brew tap farion1231/ccswitch

# 安装 CC-Switch
brew install --cask cc-switch
```

安装完成后，在「启动台」或「应用程序」文件夹中找到 CC-Switch 并启动。

![](/assets/image/CC-Switch/002.webp)

</Tab>
<Tab value="Linux">

Debian / Ubuntu 系统：

```bash
# 下载 .deb 包
wget https://github.com/farion1231/cc-switch/releases/latest/download/cc-switch_x.x.x_amd64.deb

# 安装
sudo dpkg -i cc-switch_x.x.x_amd64.deb
```

</Tab>
</Tabs>
