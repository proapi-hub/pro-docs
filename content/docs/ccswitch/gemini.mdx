---
title: "Gemini 配置"
description: "在 CC-Switch 中配置 Gemini"
---

开始前，请确认已完成[通用步骤](/ccswitch/common)中的软件下载与环境检查。

1. 打开你下载的 CC-Switch 软件，你会看到如下图的初始界面。

![image-20260514150909827](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514150909827.png)

2. 在分组条中，将分组选择至「Gemini」。

![image-20260514153855142](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514153855142.png)

3. 回顾 [创建 API 令牌](/)，在 ProAPI 中创建 **Gemini** 分组的令牌，点击复制按钮，将 API Key 复制到剪切板。

![image-20260514154025203](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514154025203.png)

4. 下拉模态框，找到「CC 切换」配置项，选择你要调用的【 Gemini 】系列模型

![image-20260514154051550](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514154051550.png)


5. 添加成功后，主界面会显示我们配置的分组。在右侧点击「启用」按钮，显示「使用中」即配置完成。

![image-20260514154258100](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514154258100.png)

![image-20260514154308498](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514154308498.png)

6. 在终端运行 `gemini`，看到对话界面并能正常回复，即表示配置完成。

![image-20260514154328498](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514154328498.png)

![image-20260514154354344](https://prorise-blog.oss-cn-guangzhou.aliyuncs.com/cover/image-20260514154354344.png)
