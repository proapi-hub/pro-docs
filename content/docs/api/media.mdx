## 端点

| 方法 | 路径 | 用途 |
| --- | --- | --- |
| `POST` | `/v1/images/generations` | 文生图 |
| `POST` | `/v1/images/edits` | 图生图 / 图片编辑（multipart） |
| `POST` | `/v1/videos` | 创建视频任务 |
| `GET`  | `/v1/videos/{id}` | 查询任务状态 |
| `GET`  | `/v1/videos/{id}/content` | 下载已完成视频 |

Base URL：`https://newapi.prorisehub.com`

鉴权头：

```http
Authorization: Bearer sk-<音视频分组令牌>
Content-Type: application/json
```

<Callout type="warn">
所有端点必须使用 **音视频模型** 分组令牌。其它分组令牌调用返回 `model not found`。
</Callout>

## 计费

- **图片**：按次计费，单价见模型表。
- **视频**：按秒计费，单价 × `seconds` × 分组倍率 (音视频模型 = 1.0)。
- 显示口径：1 USD = 1 ¥。

## 请求参数

### 通用

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | :-: | --- |
| `model` | string | ✓ | 模型名 |
| `prompt` | string | ✓ | 生成描述 |

### 图片（`/v1/images/generations`、`/v1/images/edits`）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | :-: | --- |
| `n` | integer |  | 范围 `1–4`，多数模型上限为 1 |
| `aspect_ratio` | string |  | `1:1` `16:9` `9:16` `3:2` `2:3` `4:3` `3:4` |
| `quality` | string |  | `low` `medium` `high` `auto`；兼容 `standard`→low，`hd`→medium；部分模型支持 `1k` `2k` `4k` `sd` `ultra` |
| `size` | string |  | 如 `1024x1024` `1024x1792` `1792x1024` `1024x1536` `1536x1024` `1360x1360` `2304x1296` `1296x2304` `2048x2048` |
| `image` / `image_url` | string |  | 图生图 / 编辑：参考图 URL |

### 视频（`/v1/videos`）

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | :-: | --- |
| `seconds` | **string** | ✓ | 视频时长（秒），必须传字符串：`"6"` 合法，数字 `6` 返回 `400 invalid_json` |
| `aspect_ratio` | string |  | `16:9` 或 `9:16` |
| `quality` | string |  | `low` `standard` `sd` `720p` → 720p；其它（含不传）→ 1080p。720p / 1080p 同价 |
| `image` / `image_url` | string |  | 图生视频：首帧或参考图 URL |

## 图片生成

<Tabs items={['cURL', 'Python', 'Node.js']}>
<Tab value="cURL">

```bash
curl https://newapi.prorisehub.com/v1/images/generations \
  -H "Authorization: Bearer sk-音视频分组令牌" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana",
    "prompt": "一只柴犬在草地上奔跑，油画风格",
    "n": 1,
    "aspect_ratio": "16:9",
    "quality": "medium"
  }'
```

<Tab value="Python">

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-音视频分组令牌",
    base_url="https://newapi.prorisehub.com/v1",
    timeout=240,
)

resp = client.images.generate(
    model="nano-banana-pro",
    prompt="一只柴犬在富士山下奔跑，浮世绘风格",

    extra_body={"aspect_ratio": "16:9", "quality": "high"},
)
print(resp.data[0].url)
```

## 视频生成

视频走异步任务：**创建任务 → 轮询状态 → 取下载链接**。

### 创建任务

```bash
curl https://newapi.prorisehub.com/v1/videos \
  -H "Authorization: Bearer sk-音视频分组令牌" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo-3.1-fast",
    "prompt": "一只海豚在落日海面跃出水面",
    "seconds": "6",
    "aspect_ratio": "16:9",
    "quality": "720p"
  }'
```

响应：

```json
{
  "id": "task_0TKFfjlkTrF11vvAkZtgHgxxbxqdBc4V",
  "task_id": "task_0TKFfjlkTrF11vvAkZtgHgxxbxqdBc4V",
  "object": "video",
  "model": "veo-3.1-fast",
  "status": "queued",
  "progress": 0,
  "seconds": "6",
  "size": "1280x720",
  "created_at": 1779970763
}
```

### 查询状态

```bash
curl https://newapi.prorisehub.com/v1/videos/task_0TKFfjlkTrF11vvAkZtgHgxxbxqdBc4V \
  -H "Authorization: Bearer sk-音视频分组令牌"
```

响应：

```json
{
  "id": "task_0TKFfjlkTrF11vvAkZtgHgxxbxqdBc4V",
  "object": "video",
  "model": "veo-3.1-fast",
  "status": "completed",
  "progress": 100,
  "url": "https://adobe.prorisehub.com/generated/xxxxxxxx.mp4",
  "size": "1280x720",
  "seconds": "6",
  "size_bytes": 2832606,
  "created_at": 1779970763,
  "completed_at": 1779970867
}
```

完成后 `url` 字段即 mp4 直链，可直接交给浏览器或播放器；也可经由下方端点拉取：

```bash
curl -L https://newapi.prorisehub.com/v1/videos/task_xxx/content \
  -H "Authorization: Bearer sk-音视频分组令牌" \
  -o output.mp4
```

### 任务状态

| `status` | 含义 |
| --- | --- |
| `queued` | 已排队 |
| `in_progress` | 生成中（`progress` 0–100） |
| `completed` | 已完成，可用 `url` 或 `/content` 下载 |
| `failed` | 失败，无扣费回滚 |

### 超时建议

视频生成通常需要 30 秒至 3 分钟。客户端轮询间隔建议 5–10 秒，单次请求超时 ≥ 5 分钟。
